1. What does the server in the client server architecture do?

The server is a software running on a computer, and it responds to client requests for Web pages. The server also retrieves and then transmits the Web page files to the client.

2. What does the client do?

The user/client accesses the Web via a software, called a browser. The browser then locates and display information from the Web.

Communication works through a transmission language or a protocol, like HTTP. The user requests the webpage through the browser, which speaks to the server.

The browser waits for the Webpage to be delivered, usually as a text file written in HTML. The browser then displays the paged in HTML

3. What is an IP address?

An IP address is a unique address to communicate across the Internet. It can have up to 12 digits and four parts where every part is a number between 0 to 255

4. What is DNS?

DNS stands for Doman Name Server. DNS machines are responsible for providing the IP mapping and for the upkeep of the database as new machines and IPs are added to the internet.
